## # II. Количество организаций, специальностей и (или) направлений подготовки для одновременного поступления на обучение по программам бакалавриата и программам специалитета
**Тезисный конспект раздела II. Количество организаций, специальностей и (или) направлений подготовки для одновременного поступления на обучение по программам бакалавриата и программам специалитета**

- **Статья 18.**  
  - Предельное число организаций, в которые студент может одновременно поступать на обучение по программам бакалавриата и специалитета, равно **5**.

- **Статья 19.**  
  - В каждой организации предельное число специальностей и (или) направлений подготовки, по которым студент может одновременно участвовать в конкурсах по программам бакалавриата и специалитета, определяется следующим образом:  
    - **5** – если в организации проводится прием на обучение по **5 или более** специальностям/направлениям;  
    - **число, равное фактическому количеству** специальностей/направлений, если их **меньше 5**.

- **Статья 20.**  
  - В рамках одной организации студент может одновременно подавать заявки на обучение по разным конкурсным группам, относящимся к одной и той же специальности или направлению подготовки.

- **Статья 21.**  
  - Участие студента в многопрофильном конкурсе (или конкурсах) по конкретной совокупности специальностей и (или) направлений подготовки считается как участие в конкурсе по **одной** специальности или направлению подготовки.

## # III. Вступительные испытания
**Краткий тезисный конспект раздела III. Вступительные испытания**

1. **Общие принципы приёма**  
   - Приём на обучение по программам бакалавриата и специалитета осуществляется по результатам ЕГЭ (стобалльная шкала) и/или внутренних вступительных испытаний.  
   - Приём без вступительных испытаний возможен в соответствии с частями 4 и 12 ст. 71 ФЗ № 273‑ФЗ (целевые/особые квоты, бюджетные/платные места).  
   - Для отдельных квот (пункт 66 Порядка) допускается приём без вступительных испытаний, за исключением творческих и/или профессиональных дополнительных испытаний.  
   - По программам магистратуры приём осуществляется только по внутренним вступительным испытаниям.

2. **Установление перечня и приоритетности испытаний**  
   - Организация определяет перечень вступительных испытаний и их приоритетность для ранжирования списков поступающих.  
   - Для бакалавриата/специалитета разрабатывается отдельный перечень испытаний для:  
     - лиц, поступающих из среднего общего образования;  
     - лиц, поступающих из среднего профессионального или высшего образования (профессиональное образование).

3. **Общеобразовательные испытания**  
   - Для поступающих из среднего общего образования устанавливаются:  
     - общеобразовательные вступительные испытания по предметам ЕГЭ;  
     - дополнительные испытания, предусмотренные настоящей главой.  
   - Перечень общеобразовательных испытаний согласуется с федеральным перечнем (Минобрнауки РФ, часть 8 ст. 55 ФЗ № 273‑ФЗ).  
   - Вступительные испытания по русскому языку (вступительное испытание‑1) и 2–4 профильные испытания (вступительные испытания‑2…‑4) формируются в соответствии с разделом 2 федерального перечня.

4. **Профильные испытания**  
   - Вступительные испытания‑2, ‑3, ‑4 назначаются по отдельным предметам из федерального перечня.  
   - Предмет, использованный для испытания‑2, не может применяться для испытаний‑3 и ‑4.  
   - При наличии нескольких предметов в испытании‑3/‑4 – это предметы по выбору.  
   - При выборе иностранного языка организация определяет, какие языки допускаются для ЕГЭ и/или внутренних испытаний.

5. **Дополнительные испытания**  
   - Для среднего общего образования возможны:  
     - профильные испытания по одному предмету (часть 8 ст. 70 ФЗ № 273‑ФЗ);  
     - 1–3 творческие и/или профессиональные испытания (части 9–10 ст. 70 ФЗ № 273‑ФЗ).  
   - В областях искусств при наличии творческих испытаний проводятся предварительные прослушивания.  
   - При наличии творческих испытаний в перечень включаются испытания‑1 и ‑2, а также по решению организации ‑3 и ‑4.

6. **Внутренние испытания по профессиональному образованию**  
   - Организация самостоятельно определяет форму и содержание испытаний, равное количеству общеобразовательных испытаний.  
   - Дополнительные испытания, предусмотренные для среднего общего образования, также могут применяться.  
   - Для среднего профессионального образования возможны внутренние испытания, совпадающие по содержанию с общеобразовательными, либо ЕГЭ без внутреннего испытания.

7. **Права и возможности поступающих**  
   - Поступающие из разных образовательных уровней (среднее общее, среднее профессиональное, высшее, иностранные) имеют право сдавать внутренние общеобразовательные испытания в соответствии с квотами (отдельная, особая, целевая, бюджетные, платные).  
   - Граждане Беларуси могут использовать результаты централизованного тестирования/экзамена, если не сдавали ЕГЭ по соответствующему предмету.  
   - При наличии результатов ЕГЭ и/или внутренних испытаний учитывается наивысший балл, при условии достижения минимального порога.

8. **Многопрофильные и отдельные конкурсы**  
   - При многопрофильном конкурсе организация устанавливает профильные испытания, дополнительные испытания и испытания по среднему профессиональному образованию, но не распространяется на отдельный конкурс по профессиональному образованию.

9. **Оценка и результаты**  
   - Максимальный балл за каждое испытание = 100.  
   - Минимальный балл для общеобразовательного испытания соответствует минимальному баллу ЕГЭ (часть 3 ст. 70 ФЗ № 273‑ФЗ).  
   - Минимальный балл для дополнительных и профессиональных испытаний устанавливается организацией.  
   - Результаты действительны только для учебного года приёма.  
   - Внутренние испытания проводятся очно и/или дистанционно (с идентификацией).  
   - Итоги публикуются на официальном сайте в течение трёх рабочих дней после проведения, но не позднее одного дня до публикации конкурсных списков.

10. **Порядок сдачи и апелляций**  
    - Поступающий сдаёт каждое испытание один раз (по каждому выбранному предмету).  
    - При уважительной причине допускается сдача в другой группе или резервный день.  
    - При нарушении правил составляется акт о нарушении и о непрохождении; при очном испытании – удаление с места проведения.  
    - Поступающий имеет право ознакомиться с результатами в день объявления или в течение следующего рабочего дня.  
    - Возможна апелляция о нарушении порядка или несогласии с оценкой; правила апелляций устанавливаются организацией.

## # IV. Учет индивидуальных достижений поступающих
**IV. Учет индивидуальных достиений поступающих**  

**Статья 54. Способы учёта**  
1. **Баллы за общие индивидуальные достижения** – включаются в конкурсные баллы; максимум 10 баллов (бакалавриат, специалитет).  
2. **Баллы за целевые индивидуальные достижения** – начисляются за участие в профориентационных мероприятиях, предусмотренных пунктом 150 Порядка; максимум 5 баллов (в пределах целевой квоты).  
3. **Учёт при равенстве** – при совпадении остальных критериев организации могут использовать индивидуальные достижения для уточнения ранжирования.

**Статья 55. Перечень общих индивидуальных достиений (бакалавриат, специалитет)**  

| № | Достижение | Условие начисления баллов |
|---|------------|---------------------------|
| 1 | Документы об образовании с отличием (аттестат, диплом, медали) | За каждый документ – баллы в пределах установленного лимита. |
| 2 | Участие/результаты в: <br>• олимпиадах школьников (федеральный порядок); <br>• иных интеллектуальных, творческих, физкультурных и спортивных конкурсах (ст. 77 ФЗ 273) | При отсутствии особых прав по конкретным конкурсным группам. |
| 3 | Военная служба (призыв, контракт, мобилизация) | Баллы начисляются за факт службы. |
| 4 | Деятельность в добровольных формированиях в рамках спецоперации (Украина, ДНР, ЛНР, Запорожье, Херсон) | Баллы за участие в указанных формированиях. |
| 5 | Знак отличия ГТО (золото, серебро, бронза) | По порядку награждения (приказ Минспорта № 16 от 14.01.2016). |
| 6 | Иные спортивные достижения | Перечень определяется организацией. |
| 7 | Дополнительное образование, соответствующее конкурсному профилю | По решению организации (с указанием срока/объёма). |
| 8 | Волонтёрская (добровольческая) деятельность | При соответствии установленным организацией критериям. |
| 9 | Оценка итогового сочинения (условие допуска к гос. итоговой аттестации) | По результатам проверки. |
| 10| Статус победителя/призёра чемпионата «Абилимпикс» (национальный/международный) | При наличии соответствующего статуса (распоряжение Правительства № 312‑р от 26.02.2018). |

**Статья 56. Порядок начисления баллов**  
- **Аттестат с отличием** – дополнительно может учитываться медаль «За особые успехи в учении» I/II степени.  
- **Знак ГТО** – начисление по решению организации, при условии принадлежности к возрастной группе, в которой знак получен (текущий / предшествующий год). Подтверждение: удостоверение, официальные сайты Минспорта или ГТО, либо копия распорядительного акта, заверенная должностным лицом. Баллы за знак ГТО начисляются **однократно**.  

**Статья 57. Магистратура**  
- Перечень общих индивидуальных достиений и порядок их начисления устанавливается организацией самостоятельно.

**Статья 58. Самостоятельное установление порядка**  
- Организация самостоятельно определяет порядок учёта общих индивидуальных достиений, включая количество начисляемых баллов.

**Статья 59. Учёт при равенстве**  
- При равенстве по другим критериям организация может использовать:  
  * средний балл документа об образовании;  
  * баллы по отдельным предметам/дисциплинам;  
  * иные индивидуальные достижения.  
- Перечень может быть дополнен в ходе приёма. Порядок учёта при равенстве устанавливается организацией самостоятельно.

## # V. Особые права при приеме на обучение по программам бакалавриата и программам специалитета
**Раздел V. Особые права при приёме на обучение по программам бакалавриата и программам специалитета**

1. **Пункт 60.**  
   1.1. *Лица, указанные в части 4 статьи 71 Федерального закона № 273‑ФЗ*, имеют право на приём без вступительных испытаний.  
   1.2. *Лица, указанные в пункте 2 части 4 статьи 71 № 273‑ФЗ*, получают это право только по специальностям и/или направлениям подготовки в области физической культуры и спорта.  
   1.3. *По решению организации лиц, указанных в части 12 статьи 71 № 273‑ФЗ*, предоставляется:  
        – право на приём без вступительных испытаний;  
        – приравнивание к лицам, набравшим максимальный балл ЕГЭ по предмету, соответствующему профилю олимпиады школьников, либо к лицам, успешно прошедшим дополнительные вступительные испытания профильной, творческой и/или профессиональной направленности (части 7 и 8 статьи 70 № 273‑ФЗ). При предоставлении указанного права поступающему устанавливается наивысший результат – 100 баллов (право на 100 баллов).  
   1.4. Особые права, указанные в пункте 60, могут предоставляться одному и тому же поступающему одновременно.

2. **Пункт 61.**  
   Лицам, имеющим право на приём без вступительных испытаний в соответствии с частью 4 статьи 71 № 273‑ФЗ, в течение срока предоставления указанного права предоставляется преимущество посредством приравнивания к лицам, имеющим 100 баллов по общеобразовательному вступительному испытанию или по дополнительному вступительному испытанию (особое преимущество).

3. **Пункт 62.**  
   3.1. При приёме в рамках контрольных цифр поступающий, имеющий право на приём без вступительных испытаний в соответствии с частью 4 и/или частью 12 статьи 71 № 273‑ФЗ, использует его как единое право.  
   3.2. Это право может быть использовано только при подаче заявления в одну организацию и на одну образовательную программу (вне зависимости от количества оснований).  
   3.3. В рамках одной организации и одной программы поступающий может применить право к разным конкурсным группам.  
   3.4. Если конкурсный профиль охватывает несколько специальностей, направлений подготовки или программ, право применяется к профилю в целом.

4. **Пункт 63.**  
   Для приёма лиц, имеющих право по части 4 статьи 71 № 273‑ФЗ, организация:  
   4.1. Устанавливает соответствие конкурсных профилей (программ, специальностей, направлений подготовки, совокупностей) профилям заключительного этапа всероссийской олимпиады школьников, международных олимпиад школьников и спортивным достижениям (части 1 и 2 части 4 статьи 71 № 273‑ФЗ).  
   4.2. Устанавливает особое преимущество по предметам, совпадающим с профилями олимпиады.  
   4.3. Может установить особое преимущество по иным предметам и/или дополнительным вступительным испытаниям, соответствующим тем же профилям.

5. **Пункт 64.**  
   Для предоставления особых прав по части 12 статьи 71 № 273‑ФЗ организация:  
   5.1. Устанавливает перечень олимпиад школьников, по результатам которых предоставляются особые права, из числа олимпиад, включенных в федеральные перечни (части 3 статьи 77 № 273‑ФЗ).  
   5.2. Для каждой олимпиады в перечне:  
        – устанавливает соответствие конкурсных профилей одному или нескольким профилям олимпиады либо решает об отсутствии права;  
        – устанавливает одно или несколько общеобразовательных/дополнительных вступительных испытаний, соответствующих профилям олимпиады, для предоставления права на 100 баллов либо решает об отсутствии таких испытаний;  
        – для каждого особого права определяет:  
          * предоставляется ли право победителям либо победителям и призерам;  
          * классы, в которых должны быть получены результаты;  
          * предметы, по которым требуются результаты ЕГЭ или внутренних экзаменов (за исключением творческих олимпиад и олимпиад в области физической культуры и спорта);  
          * количество баллов (≥ 75) подтверждающих право.  
   5.3. Поступающий обязан иметь указанное количество баллов по одному из предметов, выбранных им.

6. **Пункт 65.**  
   6.1. По одному основанию, дающему право на 100 баллов, поступающий получает 100 баллов в рамках одного конкурса по одному общеобразовательному вступительному испытанию (при наличии нескольких испытаний – по выбору) либо по одному/нескольким дополнительным испытаниям.  
   6.2. Поступающий может одновременно использовать несколько оснований для получения 100 баллов, даже в рамках одного конкурса.  
   6.3. При участии в нескольких конкурсах одно и то же основание может применяться к одному и тому же вступительному испытанию (предмету) или к разным испытаниям.

7. **Пункт 66.**  
   Поступающим предоставляются:  
   7.1. Право на приём на обучение на места в пределах особой квоты (часть 5 статьи 71 № 273‑ФЗ);  
   7.2. Право на приём на обучение на места в пределах отдельной квоты (части 5.1 и 5.2 статьи 71 № 273‑ФЗ);  
   7.3. Преимущественное право зачисления (часть 9 статьи 71 № 273‑ФЗ);  
   7.4. Преимущественное право зачисления (часть 10 статьи 71 № 273‑ФЗ).

## # VI. Прием заявлений и документов
**Краткий тезисный конспект раздела VI. Прием заявлений и документов**

1. **Общие требования к заявителям**  
   1.1. Поступающие на программы бакалавриата, специалитета и магистратуры подают:  
   - заявление о приёме в рамках контрольных цифр;  
   - заявление о приёме на платные места (если желают);  
   - необходимые документы.  
   1.2. Заявления и документы можно подавать:  
   - лично в организацию;  
   - через оператора почтовой связи;  
   - в электронной форме через внутреннюю систему организации;  
   - через федеральную систему «Единый портал государственных и муниципальных услуг» (ЕПГУ).  
   1.3. При использовании ЕПГУ организация может отказаться от приёма через внутреннюю систему.

2. **Сроки приёма**  
   2.1. Организация устанавливает два ключевых срока:  
   - «день завершения приёма документов со сдачей вступительных испытаний» (для тех, кто сдаёт внутренние испытания);  
   - «день завершения приёма документов без сдачи вступительных испытаний» (для тех, кто не сдаёт испытаний, включая специальные квоты).  
   2.2. При личном приёме заявитель получает расписку.

3. **Приоритеты зачисления**  
   3.1. В заявлении указываются конкурсные группы и приоритеты зачисления.  
   3.2. Приоритеты обозначаются целыми числами, начиная с 1; высота приоритета уменьшается с ростом номера.  
   3.3. Поступающий формирует отдельные последовательности приоритетов для:  
   - контрольных цифр;  
   - платных мест.  

4. **Проверка заявлений**  
   4.1. В заявлении подтверждается подписью (или отметкой в ЕПГУ) факты:  
   - ознакомление с требованиями к достоверности сведений и подлинности документов;  
   - ознакомление с правилами приёма и документами § 55 ФЗ 273‑ФЗ;  
   - подтверждение, что это первое получение высшего образования (исключая случаи бюджетного получения);  
   - ограничение количества одновременных заявлений (≤ 5 организаций, ≤ 5 специальностей);  
   - подтверждение подачи заявления только в данную организацию при праве на приём без вступительных испытаний.  
   4.2. Указывается необходимость специальных условий при сдаче вступительных испытаний из‑за инвалидности/ОВЗ.  
   4.3. Заявление подаётся на русском языке.

5. **Внесение изменений**  
   5.1. Поступающий может изменить заявление (группы, приоритеты) до установленного срока.  
   5.2. Изменения приоритетов через ЕПГУ допускаются не чаще чем раз в 2 часы.  
   5.3. После «дня завершения приёма документов со сдачей вступительных испытаний» нельзя добавлять новые группы, требующие испытаний.

6. **Документы, необходимые для поступления**  
   6.1. **Идентификация** – паспорт РФ (или паспорт за рубежом).  
   6.2. **Образование** – документы об образовании (сроки: до «дня завершения приёма документов»).  
   6.3. **Регистрация** – подтверждение регистрации в системе индивидуального учёта (если есть).  
   6.4. **Согласие** – заявление о согласии на обработку персональных данных.  
   6.5. **Инвалидность/ОВЗ** – документ об инвалидности (срок: до «дня завершения приёма документов без сдачи испытаний»); документ об ОВЗ (срок: до «дня завершения приёма документов»).  
   6.6. **Централизованные тесты** – подтверждение прохождения теста/экзамена (до «дня завершения приёма документов без сдачи испытаний»).  
   6.7. **Особые права** – документы, подтверждающие право по частям 4, 12, 5, 5.1, 5.2, 9, 10 ст. 71 ФЗ 273‑ФЗ (срок: до «дня завершения приёма документов без сдачи испытаний»).  
   6.8. **Индивидуальные достижения** – документы по усмотрению (до «дня завершения приёма документов без сдачи испытаний»).  
   6.9. **Пункты 159‑161 Порядка** – документы (до «дня завершения приёма документов без сдачи испытаний»).  
   6.10. **Фотография** – до «дня завершения приёма документов».  
   6.11. **Иностранные документы** – перевод на русский, нотариально заверенный (или консульский), и/или легализация, если требуется.

7. **Формы представления документов**  
   7.1. Документы могут подаваться оригиналами, копиями или электронными образами; заверение копий не требуется.  
   7.2. При подаче через ЕПГУ:  
   - документ считается представленным в копии, если информация о нём подтверждена в ЕПГУ или иных государственных системах;  
   - представление оригинала/копии не требуется, но возможно.  
   7.3. Если информация о документе не подтверждена, заявитель обязан представить электронный образ через ЕПГУ/внутреннюю систему или оригинал/копию лично/через почту.  
   7.4. Документы, подтверждающие особые права по частям 5.1 и 5.2 ст. 71 ФЗ 273‑ФЗ, при отсутствии подтверждения в системах требуют оригинала/копии лично или через почту.

8. **Проверка и оформление**  
   8.1. Организация проверяет достоверность сведений и подлинность документов, обращаясь к государственным информационным системам и органам.  
   8.2. На основании полученных данных формируется личное дело поступающего в электронной и/или бумажной форме.

9. **Решения по приёму**  
   9.1. После приёма заявлений, документов и проведения внутренних вступительных испытаний организация принимает решение о допуске к участию в конкурсе.  
   9.2. При превышении количества мест в особой квоте над числом допущенных к участию, а также при превышении количества мест в отдельной квоте, в течение одного дня после «дня завершения приёма документов без сдачи испытаний» проводится перераспределение мест между квотами.  
   9.3. Избыток мест после перераспределения добавляется к основным бюджетным местам.

*Данный конспект отражает ключевые положения раздела VI в строгом юридическом стиле и сохраняет структуру исходного текста.*

## # VIII. Зачисление, подача и отзыв согласия на зачисление, заключение договора об образовании, отзыв документов, отказ от зачисления
**Тезисный конспект раздела VIII. Зачисление, подача и отзыв согласия на зачисление, заключение договора об образовании, отзыв документов, отказ от зачисления**

1. **Порядок зачисления**  
   1.1. Зачисление осуществляется по конкурсным спискам в порядке приоритетов, указанных в заявлении о приёме, до заполнения установленного количества мест.  
   1.2. Приём заявлений о приёме и согласия на зачисление начинается с дня начала приёма и заканчивается в день завершения представления согласия (срок завершения представления согласия).  

2. **Согласие на зачисление**  
   2.1. Поступающий подаёт согласие в электронном виде (ЕПГУ) либо в бумажном виде (заявление о согласии).  
   2.2. Электронное согласие может подаваться не чаще, чем один раз в 2 часа.  
   2.3. Согласие относится ко всем конкурсным группам в рамках контрольных цифр приёма по программам бакалавриата, специалитета и магистратуры.  
   2.4. При желании подать согласие в другую организацию, необходимо отозвать ранее поданное согласие.  

3. **Договор об образовании (платные места)**  
   3.1. Для зачисления на платные места поступающий заключает договор об образовании и, при необходимости, информирует организацию о его наличии.  
   3.2. Договоры заключаются в период с начала приёма заявлений до дня завершения заключения договоров (включительно).  

4. **Приоритеты зачисления**  
   4.1. **Основной высший приоритет** – самый высокий приоритет, применяемый к поступающим, включённым в конкурсный список, независимо от согласия.  
   4.2. **Высший проходной приоритет** – самый высокий приоритет, применяемый к поступающим, представившим согласие.  
   4.3. Приоритеты определяются на основании всех конкурсных списков, в которых участвует поступающий.  

5. **Условия зачисления**  
   5.1. Поступающий зачисляется по высшему проходному приоритету, если:  
       - он проходит конкурс в пределах установленного количества мест;  
       - в срок завершения представления согласия имеется согласие;  
       - до издания приказа о зачислении он не отозвал согласие.  
   5.2. Платные места зачисляются, если:  
       - он проходит конкурс в пределах количества мест;  
       - в день завершения заключения договоров имеется заключённый договор;  
       - он проинформировал организацию о необходимости зачисления по договору (если требуется).  
   5.3. Зачисление на платные места осуществляется независимо от зачисления по контрольным цифрам.  

6. **Подача согласия и его отозвание**  
   6.1. Поступающий может подавать согласие через ЕПГУ, лично или через оператора почтовой связи, в зависимости от способа подачи заявления о приёме.  
   6.2. При представлении согласия организация вносит сведения о нём в конкурсный список до публикации.  
   6.3. Поступающий имеет право отозвать согласие в любой момент, подав заявление об отзыве через ЕПГУ или лично/через оператора.  
   6.4. При отзыве согласия сведения об отзыве вносятся в конкурсный список.  

7. **Отзыв заявления о приёме**  
   7.1. Поступающий может отозвать заявление о приёме в любой момент, подав заявление об отзыве через ЕПГУ или лично/через оператора.  
   7.2. При отзыве заявлений организация исключает поступающего из всех списков (подавших заявление, конкурсных списков, зачисленных).  

8. **Отказ от зачисления**  
   8.1. Зачисленный поступающий может отказаться от зачисления без отзыва согласия, подав отказ через ЕПГУ или лично/через оператора.  
   8.2. При отказе организация исключает его из числа зачисленных и корректирует конкурсные списки.  
   8.3. Если зачисленный на контрольные места отозвал согласие, он обязан одновременно отказаться от зачисления.  

9. **Сроки обработки**  
   9.1. При получении сведений с ЕПГУ или личного заявления не менее чем за 2 часа до конца рабочего дня – обработка в течение 2 часов.  
   9.2. При получении менее чем за 2 часа до конца рабочего дня – обработка в течение первых двух часов следующего рабочего дня.  
   9.3. При получении через оператора почтовой связи или ИИС – до следующего рабочего дня.  
   9.4. В день завершения представления согласия действия по пунктам 100‑103 выполняются не позднее 14:00 по московскому времени.  

10. **Возврат документов**  
    10.1. После завершения приёма оригиналы документов возвращаются в течение одного рабочего дня после заявления об отзыве.  
    10.2. При невозможности возврата оригиналы остаются в хранении.  

11. **Оформление и публикация зачисления**  
    11.1. Зачисление оформляется приказом(ами) о зачислении, публикуемыми на официальном сайте с уникальным кодом поступающего.  
    11.2. Организация публикует сведения о зачислении (код, баллы, основания для приёма без вступительных испытаний) в день издания приказов и делает их доступными в течение 6 месяцев.

## # IX. Зачисление на места в рамках контрольных цифр приема
**IX. Зачисление на места в рамках контрольных цифр приема**  

**1. Этапы зачисления**  
- **Бакалавриат и специалитет** – три этапа:  
  1) приоритетный;  
  2) основной;  
  3) дополнительный.  
- **Магистратура** – два этапа:  
  1) основной;  
  2) дополнительный.  

**2. Приоритеты на каждом этапе**  
- На каждом этапе организация определяет:  
  - **Основной высший приоритет**;  
  - **Высший проходной приоритет**.  

**3. Приоритетный этап (бакалавриат / специалитет)**  
3.1. **Зачисление**  
- **а)** основные бюджетные места без вступительных испытаний (ч. 4 и/или ч. 12 ст. 71 ФЗ № 273‑ФЗ);  
- **б)** места особой квоты:  
  - без вступительных испытаний (ч. 4/ч. 12 ст. 71 ФЗ);  
  - по результатам ЕГЭ и/или внутренних испытаний;  
- **в)** места целевой квоты: аналогично пункту б);  
- **г)** места отдельной квоты:  
  - без вступительных (кроме творческих/профессиональных дополнительных);  
  - по результатам ЕГЭ и/или внутренних испытаний.  

3.2. **Распределение по высшему проходному приоритету**  
- Если высший проходной приоритет = целевой квота → зачисление в целевую квоту.  
- Если высший проходной приоритет = иные места → порядок зачисления:  
  1) на основные бюджетные места без испытаний;  
  2) на отдельную квоту (при отсутствии конкурса на основные места);  
  3) на особую квоту (при отсутствии конкурса на основные и отдельную квоты).  

**4. Основной этап (бакалавриат / специалитет)**  
- Зачисление на основные бюджетные места **по результатам ЕГЭ и/или внутренних испытаний** в соответствии с приоритетом иных мест.  

**5. Дополнительный этап (бакалавриат / специалитет)**  
- Зачисление на незаполненные основные бюджетные места **по результатам ЕГЭ и/или внутренних испытаний** в соответствии с приоритетом иных мест.  

**6. Перераспределение незаполненных квот**  
- Незаполненные/освобождённые места особой, отдельной, целевой квот после приоритетного этапа **добавляются к основным бюджетным местам**.  

**7. Исключение из конкурсных списков**  
- Абитуриенты, зачисленные в особую или отдельную квоту, **исключаются** из списков на основные бюджетные места по тем же условиям поступления.  

**8. Отказ от зачисления и переход между этапами (бакалавриат / специалитет)**  
- **Отказ от приоритетного этапа**:  
  - При желании участвовать в основном этапе в той же организации – необходимо отказаться от приоритетного зачисления;  
  - При отказе до срока подачи согласия – абитуриент учитывается при основном этапе;  
  - При отсутствии отказа – не допускается к основному этапу.  
- **Переход в иную организацию**: отказ от приоритетного зачисления и отзыв согласия до срока подачи согласия.  
- **Отказ от приоритетного/основного этапа при желании участвовать в дополнительном этапе** (та же организация): аналогичный порядок – отказ до срока подачи согласия, последующее учёт в дополнительном этапе; без отказа – исключение.  
- **Переход в иную организацию на дополнительный этап**: отказ от предыдущего зачисления и отзыв согласия до срока подачи согласия.  

**9. Основной этап (магистратура)**  
9.1. **Зачисление**  
- На места **целевой квоты**;  
- На **основные бюджетные места**.  
9.2. **Распределение по высшему проходному приоритету**  
- Приоритет = целевой квоты → зачисление в целевую квоту;  
- Приоритет = иные места → зачисление в основные бюджетные места.  

**10. Дополнительный этап (магистратура)**  
- Зачисление на незаполненные основные бюджетные места **в соответствии с приоритетом иных мест**.  

**11. Перераспределение незаполненных мест целевой квоты (магистратура)**  
- Незаполненные места целевой квоты **добавляются к основным бюджетным местам**.  

**12. Отказ от зачисления и переход между этапами (магистратура)**  
- При желании участвовать в дополнительном этапе в той же организации: отказ от зачисления на основном этапе до срока подачи согласия; отказ → учёт в дополнительном этапе; отсутствие отказа → исключение.  
- При желании перейти в иную организацию: отказ от зачисления на основном этапе и отзыв согласия до срока подачи согласия.  

*Примечание: все действия осуществляются в сроки, установленные нормативным актом, и подлежат документальному подтверждению (отказ, отзыв согласия).*

### Статья 126 – Публичное размещение информации о приёме на обучение

   - Особенности для инвалидов и лиц с ОВЗ.  
e) Порядок учёта индивидуальных достижений.  
f) Перечень общих индивидуальных достижений, учитываемых при приёме.  
g) Информация о предоставлении особых прав и преимуществ (бакалавриат, специалитет).  
h) Информация о необходимости (или её отсутствии) обязательного предварительного медицинского осмотра.  
i) Адреса приёма документов, почтовые и электронные адреса для взаимодействия с поступающими.  
j) Наличие общежитий для обучающихся.  
k) Количество мест по различным условиям поступления в рамках контрольных цифр (без указания квот).

**2. Дополнительные сведения (не позднее 10 апреля года приёма):**

## # XII. Особенности проведения внутренних вступительных испытаний для инвалидов и лиц с ограниченными возможностями здоровья
**Раздел XII. Особенности проведения внутренних вступительных испытаний для инвалидов и лиц с ограниченными возможностями здоровья**

- **137.** Организация обязана обеспечить условия, учитывающие психофизическое развитие, индивидуальные возможности и состояние здоровья поступающих с ограниченными возможностями здоровья (ОПЗ).  
  - Предоставляется доступная информация о порядке проведения испытаний.  

- **138.** Продолжительность внутреннего вступительного испытания для ОПЗ может быть увеличена по решению организации, но не более чем на 1,5 часа.  

- **139.** При очном проведении:
  - Обеспечивается беспрепятственный доступ к аудитории, туалетам и другим помещениям (пандусы, подъемники, поручни, расширенные дверные проемы, лифты; при отсутствии лифтов аудитория должна находиться на первом этаже).  
  - Испытания проводятся в отдельной аудитории.  
  - Ограничения по численности:  
    - Письменная форма – не более 12 человек;  
    - Устная форма – не более 6 человек.  
  - Допускается превышение указанных чисел и совместное проведение с другими поступающими, если это не создаёт трудностей.  
  - Разрешено присутствие ассистента (работник организации или привлечённый специалист) для оказания технической помощи (занять рабочее место, передвигаться, читать, оформлять задания, общаться).  

- **140.** Поступающие с ОПЗ могут пользоваться необходимыми техническими средствами в процессе сдачи испытаний.  

- **141.** Дополнительные требования в зависимости от индивидуальных особенностей:  
  1. **Слепые**  
     - Задания оформляются рельефно‑точечным шрифтом Брайля, электронным документом с специализированным ПО или зачитываются ассистентом.  
     - Письменные задания выполняются на бумаге Брайля, компьютером с ПО для слепых или надиктовываются ассистенту.  
     - При необходимости предоставляется комплект письменных принадлежностей, бумага Брайля, компьютер с ПО.  
  2. **Слабовидящие**  
     - Обеспечивается индивидуальное равномерное освещение ≥ 300 люкс.  
     - При необходимости предоставляется увеличивающее устройство (в том числе собственное).  
     - Задания и инструкции оформляются увеличенным шрифтом.  
  3. **Глухие и слабослышащие**  
     - Наличие звукоусиливающей аппаратуры коллективного пользования; при необходимости индивидуальная аппаратура.  
     - Предоставляются услуги сурдопереводчика.  
  4. **Слепоглухие**  
     - Предоставляются услуги тифлосурдопереводчика, а также требования, применяемые к слепым и глухим.  
  5. **Лица с тяжелыми нарушениями речи, глухие, слабослышащие**  
     - Устные испытания проводятся в письменной форме (дополнительные творческие/профессиональные испытания, магистратура – по решению организации).  
  6. **Лица с нарушениями опорно‑двигательного аппарата, двигательных функций верхних конечностей или отсутствием верхних конечностей**  
     - Письменные задания выполняются на компьютере с специализированным ПО или надиктовываются ассистенту.  
     - Вступительные испытания, проводимые в письменной форме, проводятся в устной форме (дополнительные творческие/профессиональные испытания, магистратура – по решению организации).

## # XIII. Особенности приема на целевое обучение
**Краткий тезисный конспект раздела XIII «Особенности приема на целевое обучение»**

**142.** Приём на места в пределах целевой квоты осуществляется в соответствии с:
- Положением о целевом обучении;
- Типовой формой договора о целевом обучении, утверждёнными Правительством РФ (Пост. № 555 от 27.04.2024).

**143.** Основания приёма:
- Заявки поступающих, поданные в ответ на предложения о заключении договоров о целевом обучении;
- Предложения формируются федеральными, региональными и муниципальными органами, а также юридическими лицами, указанных в ч. 1 ст. 71.1 ФЗ № 273‑ФЗ;
- Предложения размещаются на единой цифровой платформе «Работа в России» (Пост. № 867 от 13.05.2022) либо передаются организации напрямую (при отсутствии размещения);
- Организация присваивает заказчикам, представившим предложения вне платформы, уникальные идентификационные номера.

**144.** Порядок подачи заявления поступающего:
1. Подаёт заявление в соответствии с конкретным предложением.
2. Указывает в заявлении:
   - **а)** Признак размещения на платформе «Работа в России» и номер предложения (если размещено);
   - **б)** Признак неразмещения и номер предложения, сформированный заказчиком (если не размещено).
3. При подаче новой заявки предыдущая считается отозванной.

**145.** Проверка соответствия заявления предложению:
- Организация сверяет заявление с предложением (на платформе или от заказчика);
- При отсутствии предложения либо несоответствии конкурсной группы заявка отклоняется.

**146.** В списках заявителей и конкурсных списках указываются:
- Признак (не)размещения предложения;
- Номер предложения;
- Идентификационный номер заказчика (при неразмещении).

**147.** Формирование целевой квоты:
- Квота по специальности/направлению определяется в порядке, утверждённом правительством, субъектом РФ или муниципалитетом (ч. 3 ст. 71.1 ФЗ № 273‑ФЗ, Пост. № 555 от 27.04.2024);
- Квота может быть недетализированной или детализированной в интересах конкретных заказчиков.

**148.** При детализированных квотах:
- Для каждой детализированной квоты проводится отдельный конкурс;
- Неуказанные места (без конкретного заказчика) могут использоваться для конкурсов в интересах иных заказчиков;
- Поступающий участвует в конкурсе только по одной детализированной квоте по данной специальности/направлению.

**149.** Незаполненные (освободившиеся) места детализированных квот переходят в общий фонд бюджетных мест.

**150.** Участие в профориентационных мероприятиях:
- Поступающий, прошедший такие мероприятия, указывает это в заявлении;
- Организация учитывает баллы за целевые индивидуальные достижения, если заявитель включён в список участников, представленный заказчиком до начала приёма заявлений (в бумажной или электронной форме, с подписью уполномоченного лица).

**151.** Зачисление:
- Количество зачисленных не превышает количества договоров, указанных заказчиками в их предложениях (по каждому предложению отдельно).

**152.** Приём в интересах безопасности государства:
- Приём осуществляется только при наличии информации о поступающих от соответствующих заказчиков;
- Сведения о поступающих не публикуются в открытых источниках (сайт организации, ЕПГУ, списки заявителей, результаты вступительных испытаний, конкурсные списки, сведения о зачислении);
- Оформление зачисления производится отдельным (или несколькими) приказом(ами).

## # XIV. Особенности проведения дополнительного приема на обучение
**Краткий тезисный конспект раздела XIV «Особенности проведения дополнительного приема на обучение»**

**153. Порядок проведения дополнительного приема**  
1. **Однократный приём на места в рамках контрольных цифр** – допускается, если в конкретном конкурсном профиле и форме обучения в организации (или филиале) остаётся менее 10 незаполненных мест; решение принимает организация.  
2. **Однократный приём на платные места** – осуществляется по решению организации для абитуриентов, поступающих на основании документа иностранного государства об образовании.  
3. Приём по бакалавриату и специалитету проводится независимо от приёма по магистратуре.  
4. В ходе дополнительного приёма **не производится дополнительное зачисление** (зачисление происходит только в рамках основного приёма).

**154. Содержание дополнительного приёма**  
- Организация принимает заявления и документы, проводит внутренние вступительные испытания и осуществляет зачисление.  
- По решению организации учитываются результаты внутренних вступительных испытаний, сданных абитуриентами в период основного приёма.

**155. Категории мест, включаемых в дополнительный приём в рамках контрольных цифр**  
- **Особая квота** (бакалавриат, специалитет).  
- **Отдельная квота** (бакалавриат, специалитет).  
- **Основные бюджетные места**.  

*Перераспределение мест* (до начала приёма заявлений):  
- Возможна только внутри одной **укрупнённой группы** (специальности, направления подготовки, образовательные программы/профили), для которых объявлен приём в рамках контрольных цифр.  
- Перемещение мест **не допускается** между формами обучения, между организациями (без учёта филиалов) и между филиалами.

**156. Особенности приёма по бакалавриату и специалитету**  

1. **Контрольные цифры приёма**  
   - **Особая квота** – минимум 10 % от объёма контрольных цифр, объявленных для дополнительного приёма.  
   - **Отдельная квота** – минимум 10 % от того же объёма.  
   - Абитуриент, имеющий право на приём без вступительных испытаний (ч. 4 и/или ч. 12 ст. 71 ФЗ № 273‑ФЗ), может воспользоваться этим правом в дополнительном приёме независимо от его использования в основном приёме.  

2. **Учёт организаций**  
   - В число организаций, в которых абитуриент может одновременно подавать документы, включаются:  
     a) организации, где он участвует в дополнительном приёме;  
     b) организации, где он в тот же период участвует в основном приёме на платные места.  

3. **Учёт специальностей/направлений подготовки**  
   - В количество одновременно выбираемых специальностей/направлений включаются:  
     a) те, по которым абитуриент участвует в дополнительном приёме;  
     b) те, по которым он одновременно участвует в основном приёме на платные места.  

---  
*Примечание:* Все указанные положения действуют в рамках нормативно‑правового акта и регулируют порядок, условия и ограничения проведения дополнительного приёма на обучение в высших учебных заведениях.

## # XV. Особенности приема на обучение иностранных граждан и лиц без гражданства
**Краткий тезисный конспект раздела XV. Особенности приема на обучение иностранных граждан и лиц без гражданства**

1. **Право на бюджетное обучение**  
   1.1. Иностранные граждане и лица без гражданства имеют право на получение высшего образования за счёт бюджетных ассигнований в соответствии с международными договорами РФ, федеральными законами или установленной Правительством РФ квотой (постановление № 2150 от 18 дек. 2020 г.).  
   1.2. Приём в пределах квоты осуществляется в соответствии с направлениями федерального органа исполнительной власти, осуществляющего функции по выработке государственной политики и нормативно‑правовому регулированию в сфере высшего образования (Пункт 18 Порядка отбора иностранных граждан и лиц без гражданства).  
   1.3. Зачисление в пределах квоты оформляется отдельным приказом (приказами) организации.

2. **Документы, подтверждающие статус**  
   2.1. Иностранные граждане, поступающие по международным договорам, должны представить, помимо документов, указанных в пункте 77 Порядка, документы, подтверждающие их отнесение к числу лиц, указанных в соответствующих международных договорах.  
   2.2. Соотечественники (не являющиеся гражданами РФ) должны представить оригиналы или копии документов, предусмотренных статьей 17 ФЗ № 99‑ФЗ от 24 мая 1999 г.  
   2.3. При подаче документов иностранный гражданин или лицо без гражданства обязано предоставить оригинал или копию документа, удостоверяющего личность и гражданство (или документ, удостоверяющий личность иностранного гражданина в РФ), в соответствии со статьей 10 ФЗ № 115‑ФЗ от 25 июля 2002 г.

3. **Особые условия для соотечественников**  
   3.1. Соотечественники, не являющиеся гражданами РФ, не имеют особых прав при приёме на обучение по программам бакалавриата и специалитета, предоставляемым отдельным категориям граждан РФ по ФЗ № 273‑ФЗ, за исключением права, предусмотренного пунктом 1 части 4 статьи 71 ФЗ № 273‑ФЗ, если иное не предусмотрено международным договором РФ (часть 6.1 статьи 17 ФЗ № 99‑ФЗ).

4. **Обучение по программам с государственной тайной**  
   4.1. Приём иностранных граждан и лиц без гражданства на программы, содержащие сведения, составляющие государственную тайну, осуществляется в пределах квоты с соблюдением требований законодательства РФ о государственной тайне.

5. **Вступительные испытания и конкурсные группы**  
   5.1. При приёме на платные места по программам бакалавриата и специалитета организация может самостоятельно установить перечень вступительных испытаний для иностранных граждан и минимальное количество баллов.  
   5.2. Если количество испытаний для иностранных граждан отличается от количества, установленного в пунктах 24, 34, 35 Порядка, организация выделяет отдельную конкурсную группу для иностранных граждан и формирует отдельную конкурсную группу.  
   5.3. Иностранные граждане и лица без гражданства могут выбрать поступление по результатам вступительных испытаний для иностранных граждан либо по результатам испытаний, предусмотренных пунктами 24, 34, 35 Порядка.  
   5.4. Для программ, реализуемых на русском языке с первого курса, в перечень вступительных испытаний включается испытание по русскому языку.  
   5.5. Организация вправе учитывать в качестве результата испытания по русскому языку сертификат о прохождении государственного тестирования по русскому языку как иностранному, подтверждающий владение русским языком не ниже первого уровня (Постановление № 412 от 25 апр. 2014 г. и Постановление № 255 от 1 апр. 2014 г.).

6. **Приложение**  
   6.1. Приложение N 2 к приказу Министерства науки и высшего образования РФ от 27 ноября 2024 г. № 821 содержит дополнительные уточнения, применяемые к вышеизложенным положениям.
